# Description: Created a simple webpage using Bootstrap 4 that demonstrates the use of nested media objects. The page includes a parent media object with an image and accompanying text, and within the parent media body, there is a nested media object that also contains an image and text. The images are aligned to the left of the text using the `mr-3` class for margin. The design is clean and responsive, utilizing Bootstrap's built-in classes to ensure proper spacing and alignment. The necessary libraries (jQuery, Popper.js, and Bootstrap JS) are included in the page to enable any potential interactive features of the media objects.
